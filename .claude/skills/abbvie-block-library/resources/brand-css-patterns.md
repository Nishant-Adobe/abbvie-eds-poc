# Brand CSS Patterns Quick Reference

## Button Border-Radius by Brand
- Skyrizi: `6px` (rounded rect) / `10px` (tertiary)
- Rinvoq HCP/DTC: `100px` (pill)
- Linzess: `16px` (rounded)
- Venclexta: `100px` (pill)
- Mavyret: `30px` (rounded pill)

## Footer Background by Brand
- Skyrizi: `#003b52` (dark teal)
- Rinvoq: `#17181b` (near-black)
- Linzess: `#422e83` (purple)
- Venclexta: `#000` (black)
- Mavyret: `#071d49` (dark navy)

## Hero Gradient by Brand
- Skyrizi: `linear-gradient(to top, rgba(255,255,255,0), rgb(243,243,243))`
- Linzess: `linear-gradient(rgba(0,0,0,0.8), rgba(255,255,255,0))`
- Mavyret: `radial-gradient(at 50% 100%, rgb(81,98,142) 0, rgba(7,29,73,0.9) 65%)`

## Safety Bar by Brand
- Skyrizi: White bg, `#999` border, `5px` radius
- Rinvoq: `#f5f5f5` bg, gold `#ffd100` border top
- Linzess: White bg, `16px 16px 0 0` radius, shadow
- Venclexta: `#f5f5f5` bg, gold border top
- Mavyret: `#0c2556` dark navy bg, white text

## Section Background Styles
Common across brands — map to EDS Section metadata `style` values:
- `gradient` — Brand-specific gradient (fiesta, brushstroke, etc.)
- `grey-bg` / `glacier-bg` — Neutral light grey
- `dark-bg` — Dark brand color (varies per brand)
- `indication` — Indication statement bar
- `off-white-bg` — Slightly off-white (Linzess: `#f4f6fb`)
- `teal-bg` — Venclexta teal container
- `navy-bg` — Mavyret navy
- `purple-bg` — Linzess purple
